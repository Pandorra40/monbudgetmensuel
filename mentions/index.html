Hébergement

Ce site est hébergé sur un serveur dédié. Les fichiers sont statiques et ne contiennent aucune donnée utilisateur.

Données personnelles & RGPD

Budget Mensuel ne collecte aucune donnée personnelle. Toutes les données saisies (revenus, dépenses, montants) sont stockées exclusivement dans le localStorage de votre navigateur, sur votre propre appareil.

Ces données ne sont jamais envoyées sur un serveur, ne sont jamais partagées avec des tiers, et ne font l'objet d'aucun traitement externe.

Aucun cookie de tracking, aucune publicité, aucun outil d'analyse n'est utilisé sur ce site.

Propriété intellectuelle

Le code source de Budget Mensuel est open source. Le design et le contenu éditorial sont la propriété de l'éditeur.
